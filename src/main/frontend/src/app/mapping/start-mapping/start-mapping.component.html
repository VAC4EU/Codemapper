Start the mapping from the concepts in an event definition form (EDF). Enter an EDF on the left and click "Search concepts". This will identify medical concepts in the EDF and list them on the right. Select the appropriate concepts in the list and click "Start with selected concepts" to start mapping, or click "Edit text" to modify the EDF.

Start the mapping from an existing codelist. Click on "Select file" and select a codelist on your computer. Then click "Import codelist". The mapping will contain all concepts that are associated with the codes in the codelist. Codes that are associated with the concepts but which not part of the imported codelist are disabled.
